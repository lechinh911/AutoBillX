Các chế độ chi tiết
1
Trích xuất từ Thư mục PDF / Ảnh
Khi nào dùng? Khi bạn có thư mục chứa các file hóa đơn gốc cần xử lý từ đầu.
1
Nhấn nút Trích xuất từ Thư mục PDF / Ảnh
2
Nhấn Chọn nguồn → Trỏ đến thư mục chứa hóa đơn
3
Nhấn Chọn đầu ra → Trỏ đến thư mục lưu kết quả
4
Nhấn ▶ Khởi chạy hệ thống → Chờ hoàn tất
📁 Kết quả: Bangkenhienlieu.xlsx + thư mục Hoadon.PDF/
2
Nạp danh sách từ Bảng kê Excel
Khi nào dùng? Khi bạn đã có sẵn file Excel chứa danh sách mã tra cứu hóa đơn.
1
Nhấn nút Nạp danh sách từ Bảng kê Excel
2
Nhấn Chọn tệp → Chọn file Excel (.xlsx)
3
Nhấn Chọn đầu ra → Trỏ đến thư mục lưu kết quả
4
Nhấn ▶ Khởi chạy hệ thống → Chờ hoàn tất
📁 Kết quả: Bangkenhienlieu.xlsx + thư mục Hoadon.PDF/
💡 Chưa có file Excel? Nhấn 「📥 Tải file Bảng kê mẫu」 trên giao diện để tải template.
3
Export JPG EGOV
Khi nào dùng? Khi cần xác minh hóa đơn trên Cổng thuế GDT và lấy ảnh chụp minh chứng.
1
Nhấn nút Export JPG EGOV
2
Nhấn Chọn tệp → Chọn file Bangkenhienlieu.xlsx
3
Nhấn Chọn đầu ra → Trỏ đến thư mục lưu kết quả
4
Nhấn ▶ Khởi chạy hệ thống → Chờ hoàn tất
📁 Kết quả trong thư mục Hoadon.JPG/:
  • 1_TONG HOP HINH ANH EGOV.pdf — PDF gộp ảnh
  • 2_EGOV_Bangkenhienlieu.xlsx — Excel kết quả EGOV
  • Các file .png — Ảnh chụp từng hóa đơn
⏱ Mỗi hóa đơn cần 5–15 giây để tra cứu trên GDT. Quá trình này tốn nhiều thời gian.
